# Roadmap

This roadmap describes current foundations and likely future areas. Future areas are not fixed commitments; each should be validated through sprint goals and user value.

## Completed Foundations

- UI foundation
- Backend foundation
- Frontend-backend integration
- Video metadata pipeline
- Multi-item media library
- Project output settings
- Preview generation for video items
- Scene Detection Foundation
- Whisper transcription foundation

## Recommended Next Pipeline Order

1. Scene and speech analysis
2. Timeline segment generation
3. AI editing decisions
4. Export pipeline

## Likely Future Areas

- Final Cut Pro compatibility
- Performance hardening
- Security hardening
- Automated testing
- Production readiness

Caching, progress tracking, job infrastructure, and shared abstractions should be introduced when actual pipeline growth justifies them.

Each future area should preserve the local-first architecture unless a sprint explicitly changes that direction.
